# Project Vision
This project aims to become a complete personal finance platform.

To start of with, we need to create a central page where assets (physical items such as gold, estates or other valuables) and savings accross all different accounts can be tracked. Think of this as a reminder of where all of your assests are. To expand on this, we can also add sections for keeping track of loans outgoing and incoming and their status. Then, we can have a page tracking performances of these assets over time. 
Hopefully, we can create a bot used to trade in realtime but thats going to take a lot of time and effort to even create a theoretical model, so more on that later.
